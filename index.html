GP Assignment Universtiy Project

During my second year I was introduced to code ignitor which is a php framework that used a Model View Controller (MVC) architectural pattern. For this unit’s coursework I created a GP surgery portal that included using google charts API to analyse live patient data, session, and cookies for user account with roles and interactive forms for potential patients to submit data to. I really enjoyed completing this piece of course work and received a final grade of 95%.
