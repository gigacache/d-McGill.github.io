Agnes Beauty Atelier

Company website

Technology

OverView

Based in Cheshire, Agnes Beauty Atelier offers bespoke beauty services from permanent makeup to eyelash extensions and eyebrow treatments. After receiving the branding pack from the client, I developed a minimal and clean design to fit in with the brand. The client wanted a light and to the point design so clients could effortlessly find information without clicking through numerous pages. Although, at the beginning the client was not certain on how the website should look, I created different mock-ups of websites using different designs to narrow down the options and see which one was the best for the Beauty Salon. Additionally, responsiveness was very important as user ability was a top priority. This led to altering some design features on different screen sizes and devices. For example, making buttons larger, reducing image sizing and layout padding. On completion of the project, I received excellent feedback and fulfilled the client’s brief and expectations. This led to additional work of creating informative graphics for the client’s Social Media channels such as Instagram. As engagement and presence via Social Media is key for small business, I regularly create content on a weekly basis to ensure content is being uploaded on the client’s website daily.

Custome WordPress Theme

Technology

OverView

Based in Cheshire, The Lash Room Crewe offers bespoke eyelash extensions. My aim was to create an informative website for potential clients. I met with Agnes to discuss goals, ideas and features of the site. WordPress was chosen because of its user friendly back-end. I created an initial design and selected a theme. Agnes approved it and the site came to life. I created an informative five-page website that portrayed the true brand of the company. The home page featured a simple and easy to use panel navigation that linked to specific information. The site is fully responsive which makes it easy to browse on different types and sizes of device. The layout changes slightly to make buttons easier to press. Finally, then I bought a domain and set up an SSL certificate. The project was very enjoyable, and it was a pleasure working with The Lash Room Crewe. This project improved my time management as a launch date for the website had been discussed during the initial discussion. My creative thinking was put to the test by creating a simple user interface for the home page that would guide potential customers to the right information. I improved my use of CSS because of the extent of the theme that needed to be customised. I worked hard getting communication with the client right – nothing was missed and there were no hiccups.

Based in East Yorkshire, Pear Tree Maintenance offers a range of building and plumbing services. This was my very first independent client. While working as an apprentice web developer for a digital marketing company in Hull, I had worked on a few projects and met a few clients. This built up confidence to do a project of my own. The project was to create a website for the company. The client wanted a webpage with their services, prices and a contact number. I saved time and cost for the client by using a template and customising it to the client’s specification. I searched the web for a static HTML & CSS template. The client was pleased with the results as his expectations were a minimal and simple webpage rather than full website. I learnt a lot from this project as it was my first real client. It was a pleasure working with Pear Tree Maintenances. This project improved my confidence with taking on my own development projects and I proved to myself that I was capable of successfully delivering a project that a client loved. This was the first of many projects that I got myself involved with! Now I’m pretty confident undertaking development work.
